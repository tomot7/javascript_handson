はじめての電卓アプリ

はじめに

このハンズオンでは、JavaScriptを使って簡単な電卓アプリを作ります。順番に進めていけば、プログラミングが初めての人でも、JavaScriptの基本を学びながら、実際に動くアプリを作ることができます。

準備

まず、テキストエディタ（メモ帳、VSCode、Sublimeなど）を用意しましょう。新しいHTMLファイルを作って、以下の内容をコピーしてください。

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>はじめての電卓アプリ</title>
</head>
<body>
    <h1>はじめての電卓アプリ</h1>

    <script>
    // ここにJavaScriptのコードを書きます
    </script>
</body>
</html>

ステップ 1: 電卓の見た目を作る

まず、電卓の基本的な見た目を作ります。次のコードを<h1>タグの下に追加してください。

<input type="number" id="num1" placeholder="数字1">
<input type="number" id="num2" placeholder="数字2">
<select id="operator">
    <option value="+">+</option>
    <option value="-">-</option>
    <option value="*">×</option>
    <option value="/">÷</option>
</select>
<button onclick="calculate()">計算する</button>
<p id="result"></p>

ステップ 2: 計算の機能を作る

次に、計算を行う機能を作ります。以下のコードを<script>タグの中に書いてください。

function calculate() {
    // 入力された数字を取得する
    const num1 = parseFloat(document.getElementById('num1').value);
    const num2 = parseFloat(document.getElementById('num2').value);
    const operator = document.getElementById('operator').value;
    let result;

    // 計算する
    switch (operator) {
        case '+':
            result = num1 + num2;
            break;
        case '-':
            result = num1 - num2;
            break;
        case '*':
            result = num1 * num2;
            break;
        case '/':
            result = num1 / num2;
            break;
    }

    // 結果を表示する
    document.getElementById('result').textContent = `計算結果: ${result}`;
}

ステップ 3: JavaScriptの文法を理解しよう

ステップ2で書いたJavaScriptのコードで使われている主な文法について説明します。

1. 関数の定義

function calculate() { ... }

functionという言葉を使って、calculateという名前の関数（命令のまとまり）を作ります。

もっと詳しく知る

2. 変数の作成

const num1 = parseFloat(document.getElementById('num1').value);

constを使って、値が変わらない変数を作ります。letを使うと、後で値を変えられる変数になります。

もっと詳しく知る

3. HTMLの要素を取得する

document.getElementById('num1')

document.getElementById()を使って、HTMLの中の特定の要素を探して取得します。

もっと詳しく知る

4. 文字列を数字に変換する

parseFloat(document.getElementById('num1').value)

parseFloat()を使って、文字列（テキスト）を小数点のある数字に変換します。

もっと詳しく知る

5. 条件によって処理を変える

switch (operator) {
    case '+':
        result = num1 + num2;
        break;
    // ...
}

switch文を使って、条件（ここでは演算子）によって違う計算をします。

もっと詳しく知る

6. 文字列の中に変数を入れる

`計算結果: ${result}`

バッククォート(`)で囲んだ文字列の中に、${}を使って変数の値を入れることができます。

もっと詳しく知る
